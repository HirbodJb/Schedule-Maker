CSV Format Guide
Each tutor should be on a new line using this format. You do not need to type a header row:
Tutor Name, Email, Phone, Desired Hours, Mode, Saturday, ENG101, Priority, Mon 9:00, Mon 9:30...
Example:
Jamie Ray, jamie@laccd.edu, (818) 555-0100, 10, both, yes, yes, agree, Mon 9:00, Mon 9:30, Tue 1:00
  • Phone: optional, but recommended
  • Desired Hours: a number, like 8, 10, 12, or 24
  • Mode: both, oc, or ol
  • Saturday: yes or no
  • ENG101: yes if completed, no if not yet
  • Priority: agree or disagree
  • Separate each value with commas
  • Keep one tutor per line
